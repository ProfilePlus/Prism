# Prism 1.0.3 Windows Smoke Checklist

本清单用于 Windows 发布前的最小人工验收。目标不是覆盖全部功能，而是确认 Prism 的跨平台基础行为不依赖 macOS 假设。

## 安装与启动

- 安装包可正常安装、覆盖安装、卸载。
- 从开始菜单启动 Prism，窗口尺寸、最小尺寸、标题栏按钮正常。
- 新建窗口、关闭窗口、最小化、最大化行为符合 Windows 预期。

## 文件关联与打开

- 双击 `.md`、`.markdown`、`.txt` 文件可以交给 Prism 打开。
- 从系统“打开方式”选择 Prism 后，文件内容正确加载。
- 启动时携带文件路径时，左侧工作区自动定位到父目录。
- 启动时携带文件夹路径时，文件树只显示 Markdown / Text 文档和有效目录。

## 路径与文件操作

- 打开、保存、另存为、导出默认目录正确。
- 文件树中新建文件、新建文件夹、重命名、创建副本、删除可用。
- `复制路径` 得到 Windows 反斜杠路径时不被错误改写。
- `在资源管理器中显示` 能打开当前文件或工作区位置。
- 文件树排序按名称、修改时间、创建时间、大小生效。

## 快捷键与菜单

- 菜单、命令面板、快捷键面板显示 `Ctrl` / `Alt` / `Shift` 文案，而不是 macOS 符号。
- `Ctrl+N`、`Ctrl+O`、`Ctrl+S`、`Ctrl+F`、`Ctrl+B`、`Ctrl+I` 可用。
- `F8` 专注模式、`F9` 打字机模式、`F11` 全屏行为正常。
- 右键菜单中的文件管理器文案显示为 `资源管理器`。

## 导出

- PDF / HTML / Word / PNG 四种导出可以写入 Windows 用户目录。
- 同名文件覆盖提示使用 Prism 自定义弹窗。
- 导出包含中文、表格、代码块、Mermaid、图片时不丢内容。

## 主题与显示

- 五个主题切换无明显延迟。
- 编辑、分栏、预览的主题颜色、搜索高亮、选区颜色一致。
- Nocturne Dark 在 Windows 上没有白屏、闪烁或不可读菜单。
