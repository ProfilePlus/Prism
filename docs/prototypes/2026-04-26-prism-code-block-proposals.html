Prism 代码块样式选型

为您精选的 5 种代码块排版风格，适用于 Markdown 预览区。

GitHub Classic (专业纯净) #01
最经典、最具代码直觉的排版。浅灰背景配合极细的描边，不过分抢眼，非常适合作为默认的亮色主题代码块。 
function calculatePhysics(mass: number) {
    // Standard gravity constant
    const GRAVITY = 9.81;
    return mass * GRAVITY;
}
Mac Terminal (沉浸深色) #02
带有 macOS 标志性窗口红黄绿三圆点的深色主题（类似 One Dark）。非常适合强调代码内容的层级，在浅色长文中形成极强的视觉对比和沉浸感。 
function calculatePhysics(mass: number) {
    // Standard gravity constant
    const GRAVITY = 9.81;
    return mass * GRAVITY;
}
Notion Minimal (极致留白) #03
无边框、更紧凑的圆角、更大的内部留白（Padding）。它不像是代码编辑器，更像是文档中的一个高亮引用区，适合非重度开发者的文字工作者。 
function calculatePhysics(mass: number) {
    // Standard gravity constant
    const GRAVITY = 9.81;
    return mass * GRAVITY;
}
Nord Slate (北欧冷淡风) #04
低对比度的灰蓝色调，右上角附带悬浮的**语言标签 (Language Badge)**。这是现代技术博客（如 Vercel, Tailwind）最喜欢的排版方式，极具工程美感。 
function calculatePhysics(mass: number) {
    // Standard gravity constant
    const GRAVITY = 9.81;
    return mass * GRAVITY;
}
Fluent Neon (霓虹赛博) #05
专为 Night 主题设计的暗黑风。纯黑背景，带有一抹极具科技感的**霓虹青色左侧高亮边框**，关键字带有轻微的光晕特效（Glow）。 
function calculatePhysics(mass: number) {
    // Standard gravity constant
    const GRAVITY = 9.81;
    return mass * GRAVITY;
}
