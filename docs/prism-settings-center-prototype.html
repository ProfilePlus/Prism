预览排版

Beautiful writing, plain Markdown

字体是写作产品最敏感的部分。Prism 应允许用户导入自己下载的字体，并选择应用到编辑器、预览或两者。

代码仍使用 JetBrains Mono 或主题指定的等宽字体。

写作主题

主题不是单独配色卡，而是一套完整写作环境：编辑器、预览、搜索、导出、代码和 Mermaid 共同生效。

导出

第一版只展示已实现或接近已实现的导出偏好，不把 PDF 纸张、边距等未完成项放进正式设置。

HTML 包含主题
导出时内联当前主题样式，保证发给别人后可直接打开。
PNG 清晰度
当前正式版已实现 pngScale。

文件与保存

当前正式版只有自动保存间隔。建议后续改成“自动保存开关 + 保存策略”，实现前不放入正式设置。

当前 demo 将这一页作为产品边界说明：未实现的“默认导出位置”“恢复上次窗口”“保存策略”不进入第一版设置中心。

快捷键

1.0.3 先保留跨平台快捷键文案风格，不做完整快捷键编辑器。

快捷键文案
菜单、命令面板、快捷键面板中的显示方式。

高级

字体导入正式落地时，高级区应增加“打开字体目录”和“移除导入字体”。

字体目录
生产版建议保存在 appData/fonts，并在 config.json 中记录 family、filename、scope。
后续实现
字体导入实现建议：Tauri 用 dialog 选择字体文件，复制到 appData/fonts，使用 CSS FontFace 或动态 @font-face 注册；设置中保存 font id 和应用范围。不要只保存原始绝对路径，避免移动字体文件后失效。
