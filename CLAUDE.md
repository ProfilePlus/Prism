# Claude 工作指南

## 语言偏好

**始终使用中文与用户交流。**

- 所有对话、问题、设计讨论都用中文
- 代码注释可以用英文（遵循行业惯例）
- 文档、设计文档、规划文档都用中文
- commit message 用中文

## 项目背景

Prism 是一个 Windows 11 风格的 Markdown 桌面编辑器，基于 Tauri 2 + React + TypeScript。

当前状态：
- 核心编辑功能已实现（编辑/分栏/预览、自动保存、KaTeX、Mermaid）
- 产品方向已从多标签页调整为 Typora 式单文档单窗口
- 但与原型 `prism.html` 相比，桌面外壳、菜单系统、视觉精修尚未完成

## 工作原则

1. **使用 superpowers 工作流**
   - 任何新功能或重构都先走 brainstorming → writing-plans → executing-plans
   - 不要跳过设计阶段直接写代码

2. **原型对齐优先**
   - `prism.html` 是视觉和交互的参考标准
   - 实现时要对照原型的细节：布局、间距、颜色、动画、交互反馈

3. **产品决策权在用户**
   - 如果用户说"这个不对"或"改成这样"，立即调整
   - 不要坚持技术方案或设计理念，用户体验优先

4. **诚实汇报进度**
   - 不要说"完成了"除非真的完成了
   - 遇到问题或不确定时，明确说出来
